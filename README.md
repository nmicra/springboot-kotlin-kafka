# springboot-kotlin-kafka
demo project using springboot kotlin kafka

# Docker Compose
docker-compose up -d

# Before Starting Application in local-windows machine
1. Edit c:\Windows\System32\drivers\etc\hosts file
2. Add the following lines
   127.0.0.1 kafka1
   127.0.0.1 kafka2
   127.0.0.1 kafka3
